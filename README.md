# Static Firmware Security Analysis - TP-Link TL-WR841N

**A Comparative Study Across Hardware Revisions V10, V13 & V14**
*with STRIDE, OWASP IoT Top 10, and ETSI EN 303 645 Assessment*

---

## Overview

This repository contains a comprehensive static firmware security analysis of the **TP-Link TL-WR841N** wireless router. Three firmware versions spanning **seven years of development** (2015-2022) were systematically extracted, deconstructed, and evaluated using industry-standard reverse engineering tools and three complementary security frameworks.

## Key Findings

| Metric | Result |
|--------|--------|
| **Composite Security Score** | **2.1 / 10** |
| **STRIDE** | 6/6 threat categories affected |
| **OWASP IoT Top 10** | 9/10 categories FAILED |
| **ETSI EN 303 645** | 11/13 provisions NON-COMPLIANT |
| **Verified CVEs** | 4 (CVSS 6.1-8.8) |
